# Лабораторная работа №1: Первичное исследование и оценка качества данных

## Описание проекта
Первичный анализ датасета "Академическая успеваемость студентов" в рамках дисциплины "Интеллектуальный анализ данных".

**Предметная область:** в набор данных включены такие факторы, как количество учебных часов учащихся, посещаемость, уровень мотивации, доступ к учебным ресурсам, участие родителей, квалификация преподавателей, влияние сверстников и семейное происхождение. Также учитываются элементы окружающей среды и образа жизни, включая доступ к интернету, режим сна и тип школы. Возможные задачи интеллектуального анализа данных: прогнозирование уровня успеваемости (низкий/средний/высокий), кластеризация студентов по типам обучения и выявление ключевых индикаторов отставания для корректировки образовательных программ.  
**Цель лабораторной:** аудит качества данных, построение паспорта датасета, выявление проблем.

## Структура проекта
- `notebooks/01_data_understanding.ipynb` — основной Jupyter Notebook с анализом  
- `report/quality_report.md` — текстовый отчёт с выводами и планом улучшения  
- `data/` — датасет или скрипт загрузки
